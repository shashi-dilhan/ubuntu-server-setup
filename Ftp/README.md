# FTP Accounts & Users

## User groups

Create a user group

    $ sudo groupadd new_group_name
    
Delete user group

    $ sudo groupdel group_name
    
Add user to a group

    $ sudo adduser user_name user_group
    
Delete user from group

    $ sudo deluser user_name user_group
    
Check user groups

    $ sudo vim /etc/group
